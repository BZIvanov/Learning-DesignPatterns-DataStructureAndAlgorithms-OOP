Allows classes with incompatible interfaces to work together by wrapping its own interface around that of an already existing class.

It is about making 2 incompatible interfaces compatible.

It is not about changing the behaviour of the adaptee, only to make it compatible (usable)

To create the needed .js version of the .ts file navigate to this file folder and run: tsc adapter-pattern.ts You need to have installed typescript npm package also.
