About

The logic of the algorithm is that we assume the first element is already sorted. Then we loop through the next elements in the array one by one. If the previous element is bigger than the next we move all the next elements to the right and put the current item in between.

i 0 1 2 3 4 here are the indexes of the array
2 8 3 1 5 initial array
1 2 8 3 1 5 we start from index 1, because we assume the first element is already sorted and because 2 < 8 no changes needed
2 2 3 8 1 5 for index 2 we have 8 < 3 so sorting is needed. We place 3 before 8 and move 8 on the right
3 1 2 3 8 5 for index 3 we have 8 < 1 so sorting is needed. We place 1 before 8 and move 8 on the right, but 3 < 1 so we also move 3 on the right, same for 2
4 1 2 3 5 8 for index 4 we have 8 < 5 so sorting is needed. We place 5 before 8 and move 8 on the right
entries in blue move to the right
entries in red is the current number we move before the bigger numbers
entries in grey are unchanged positions
