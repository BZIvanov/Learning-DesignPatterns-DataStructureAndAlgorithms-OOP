About

The logic of the algorithm is that we use the last element of the array (the pivot) and create array with smaller numbers than the pivot and do the same for bigger numbers. Once we split everything except the pivot as arrays with 1 element we return the 1-element arrays concatenated with the pivot at resepctive positions.

It is called Quick sort, it works quick based on multiple conditions.

Visualization of the algorithm

2 8 7 4 3 1 5 initial array
2 4 3 1 5 8 7 we take as pivot the most right number and move smaller number on left and bigger numbers on right
1 2 4 3 7 8 with the recursion for the smaller array (left one) from the above row we again take as pivot the most left number which is 1 and move all others in the bigger array, because they are all bigger than 1. With the recursion we do the same for the bigger numbers array from the above row, where the pivot is 7
2 3 4 we keep doing the same with recursion and 3 is now the pivot. For 7 as pivot from the above row recursion completed, because now we have only the pivot and 1-element array
2 3 4 here the recursion also completed, because we ended up with the pivot only and 1-element arrays and now we climbing back the recursion stack
1 2 3 4 7 8 here we keep climbing the recursion stack
1 2 3 4 5 7 8 and now we have the final result
