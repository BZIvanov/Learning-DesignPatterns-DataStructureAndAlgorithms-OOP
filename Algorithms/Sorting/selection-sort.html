About

The logic of the algorithm is that we loop through every element and if it is the minimum value we move the value on the left appending it to the previous minimum element we processed.

It is called Selection sort, because everytime we select the minimum value and place it after the previous minimum value we found.

Visualization of the algorithm

i 0 1 2 3 4 here are the indexes of the array
2 8 3 1 5 initial array
0 1 8 3 2 5 we loop through all elements find the smallest and move it to the first position on the first itteration
1 1 2 3 8 5 for index 1 and higher now the minimum value is 2 so we swap the indexes to move number 2
2 1 2 3 8 5 3 is the minimum number, but it was already where it should be so nothing changes here
3 1 2 3 5 8 for the 3rd index now we swap 5 with 8 and finish the loop without the 4th index, because by now there we have the biggest number
entries in blue swap position with the smallest number
entries in red is the smallest number for the current itteration and we place it after the previous smallest number
entries in grey are unchanged positions
