About

Keynotes about the binary search tree algorithm:

  • The top of the tree is called 'root'
  • Binary tree is called binary, because each node has maximum of 2 branches/children
  • Leaf nodes are the ends of the tree, nodes with no children
  • Binary trees are ordered, each left subtree is less than or equal to the parent node and each right subtree is greater or equal to the parent node
