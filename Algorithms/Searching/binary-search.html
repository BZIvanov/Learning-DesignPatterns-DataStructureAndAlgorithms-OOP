About

Keynotes about the binary search algorithm:

  • It uses the divide and conquer method.
  • It is only working for sorted collections.

Visualization of the algorithm

1 3 4 5 9 23 48 51 59 83 91 this is our initial array and the number we will search for is 9
1 3 4 5 9 23 48 51 59 83 91 the middle of the array is index 5, but there is not the number we search for and the searching number is smaller than 23 so we proceed with the left side of the array
1 3 4 5 9 we again start in the middle and there we don't find the searched number and the searched number is bigger than 4 (the middle number) so we proceed with the right part of the subarray
5 9 the middle index in 2-elements array for us is the left element, but that is not the number we search for so we proceed again with the right part
9 this is the only element that is left and it is the number we search for so we return the number on index 4 in the initial array

Complexity

Space complexity is 0(1), because we don't create additional arrays, we operate on the initial one.
