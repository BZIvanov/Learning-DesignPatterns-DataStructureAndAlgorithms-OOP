About

Graph is a data structure made up of nodes or vertices and edges which are the connections between nodes.

Trees are also special kind of graphs, because for trees we need to fullfil some conditions to call them trees. Trees have one root and only one unique path between any two nodes.

Graph types

  • directed - two way connection, for example friends on facebook
  • undirected - one way connection, for example link on website get us on another page, but doesn't return us back
  • weighted - the edges matter, for example cities are nodes and roads are edges, then the distance between cityA to cityB can be different from the distance between cityB and cityC
  • unweighted - the edges doesn't matter
  • cyclic - we can go in circles and get back to a node
  • acyclic - the flow is in one direction and we can't go back
