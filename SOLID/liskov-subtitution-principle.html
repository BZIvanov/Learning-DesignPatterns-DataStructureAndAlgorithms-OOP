Definition

Substitutability is a principle stating that, if S is a subtype of T, then objects of type T may be replaced with objects of type S (i.e., an object of type T may be substituted with any object of a subtype S) without altering any of the desirable properties of the program (correctness, task performed, etc.).

Substitutability

  • child classes must not remove base class behavior
  • chlid classes must not violate base class variants
